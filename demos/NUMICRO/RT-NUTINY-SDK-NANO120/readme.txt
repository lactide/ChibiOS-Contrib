*****************************************************************************
** ChibiOS/RT port for ARM-Cortex-M0 NANO120KE3BN.                          **
*****************************************************************************

** TARGET **

The demo runs on a Nuvoton NUTINY-SDK-NANO120 board.

** The Demo **

The demo shows how to blink the LED on pin E.6 of the board.

** Build Procedure **

The demo has been tested by using the free ARM-none GCC toolchain. Just
modify the TRGT line in the makefile in order to use different GCC
toolchains.
